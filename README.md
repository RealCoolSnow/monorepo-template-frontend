# monorepo-template-frontend

## Project struct
    apps/vue3-app 
    apps/react-app
    packages/*  // common packages

## Install `pnpm` first
`npm i pnpm -g`

## Install dependencies
`pnpm i`

## Run 
`pnpm dev --filter "vue3-app"` 

## Build
`pnpm build --filter "vue3-app"`
